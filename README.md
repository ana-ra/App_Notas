<h1 align="center"> Aplicativo de Notas</h1>

   A proposta do projeto é a construção de um aplicativo que guarda notas com algumas informações tais como: data, descrição, a prioridade da nota e possiveis tarefas a serem executadas. Como ferramentas para uma melhor visualização foi desenvolvida uma ferramenta que apresenta notas com diferentes cores, para facilitar a visualização na tela principal. 
   
   O aplicativo possui quatro telas, uma que aparece vazia enquanto não há notas(1) , uma tela para a criação de notas (2), uma para edição da nota que foi criata previamente (3), e uma telva de exibição da nota selecionada (4).
